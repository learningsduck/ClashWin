## 4. 建议你本地手动验证的步骤

1. **启动前端**：在项目根目录执行 `pnpm dev`，打开应用。
2. **登录页**：未登录时应自动跳转到 `/login`。
3. **发送验证码**：输入手机号（如 13800138000），点击「获取验证码」；在 S2 上执行 `pm2 logs auth-server` 查看打印的 6 位验证码。
4. **登录**：输入验证码后点击「登录」，应跳转首页并提示「登录成功」。
5. **自动登录**：关闭应用重新打开，应直接进入首页。
6. **踢线**（可选）：在另一台设备或浏览器用同一账号登录，原设备应提示「您的账号已在其他设备登录」并跳转登录页。
