========================================
  登录页验证码 60 秒倒计时 - 运行说明
========================================

【已实现效果】
- 点击「获取验证码」发送成功后，按钮变为「60秒后重发」并每秒倒数
- 倒数期间按钮不可点击，倒数结束后可再次点击发送

【推荐运行方式】

因项目路径含中文（「源码」），在部分终端下 Node 会报错。建议任选其一：

方式一：在 Cursor 内置终端运行（推荐）
  1. 在 Cursor 中按 Ctrl+` 或菜单「终端」→「新建终端」
  2. 确认当前目录为 clash-verge-rev-dev（含 package.json）
  3. 执行：pnpm run web:dev
  4. 浏览器打开 http://localhost:5173 ，进入登录页查看效果

方式二：仅前端开发服务器（不启动 Tauri 桌面）
  若方式一仍报错，可将项目复制到无中文路径后运行，例如：
  - 复制整个 clash-verge-rev-dev 到 D:\clash-verge-dev
  - 在 D:\clash-verge-dev 下执行：pnpm install && pnpm run web:dev
  - 浏览器打开 http://localhost:5173

方式三：完整桌面应用（Tauri）
  在无中文路径下（如 D:\clash-verge-dev）执行：pnpm dev
  会编译并打开桌面窗口，从应用内进入登录页测试。

【测试验证码倒计时】
  1. 打开登录页
  2. 输入合法手机号（如 13800138000）
  3. 点击「获取验证码」
  4. 观察按钮变为「60秒后重发」并倒数，倒数结束后可再次点击
